# PHP With Telegram API Project

This project allows you to send messages to a Telegram group using a web form and a Telegram bot. It uses PHP and the Telegram Bot API to send messages.

## Getting Started

These instructions will help you set up and run the project on your own server.

### Prerequisites

- PHP installed on your server
- A Telegram Bot API token
- A Telegram group chat ID

### Installation

1. Clone the repository to your server:

   ```bash
   git clone https://github.com/hasndev/PHPWithTelegramAPI.git
   ```

2. Update the `send_message.php` file with your Telegram Bot API token and group chat ID.

3. Ensure your server is set up to serve PHP files.

4. Access the `index.php` page in your web browser to use the application.

## Usage

1. Access the web page (`index.php`) in your browser.
2. Fill in the message in the provided form.
3. Click the "Send Message" button.
4. The message will be sent to the specified Telegram group.

## Built With

- [PHP](https://www.php.net/) - The server-side scripting language used
- [Telegram Bot API](https://core.telegram.org/bots/api) - API used for sending messages to Telegram
